# React-Typescript 프로젝트 템플릿

## Installation

0. Project Fork

1. Node Version Setting(V22.11.0)

```
nvm install
nvm use
```

2. Install pnpm

```
npm install -g pnpm
```

2. Install Dependencies

```
pnpm install
```

3. Run

```
pnpm run dev
```

## Dependency List

- Vite
- TailwindCSS
- Prettier
- Shadcn/ui
- zustand
